Описание программы.

Данная программа была разработана на языке Python (v.3.6.9) на базе фрейморка Django (v. 3.2.6).
В ней была реализована автоматизация документооборота. Также для создания графического интерфейса былы использованы HTML5/CSS3 и библиотека Bootstrap 4.
А именно:

  • Автоматическое проставление даты добавления и изменения документа.
  • Автоматическое регулирование пользователя создавшего/изменившего документ.
  • Фильтр/Поиск документов

Сверху расположена навигационная панель.
Для получения доступа ко всем инструментам приложения необходима авторизация. После чего появляются дополнительные ссылки на навигациооной панели. Перейдя на страницу документы вы попадете на страницу с управлением документов. Там их можно удалять, изменять, добавлять и производить поиск по базе данных документов.
Ниже строки с кнопкой добавления документов, располагается таблица всех документов в базе данных. Столбец "Файл" является ссылкой на загруженный документ, нажав на нее, можно скачать документ.

У пользователей есть права доступа, по ссылке можно создавать новых пользователей, но создавать их может только пользователь с правами директора. По ссылке административная панель можно попасть в стандартную административную панель. Туда может попасть только пользователь с активным "статусом персонала". Его лучше активировать для директора и системного администратора. Для других пользователей можно оставить выключенным.
Приведенная административная панель является стандартной панелью Django. В ней пользователь получает доступ как к документам, так и к пользователям и может их редактировать и удалять.
